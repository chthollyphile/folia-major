# Stage Client Demo

这个目录提供一个最小 Stage 联调客户端页面。

运行方式：

```bash
npm run stage:client
```

然后在浏览器打开的页面中：

1. 填入 Folia Stage 的 Bearer token
2. 选择音频 URL 或音频文件
3. 提供标准 LRC 或增强 LRC
4. 点击 `Push Session`

内置样例素材：

- `fixtures/stage-demo-tone.wav`
- `fixtures/stage-demo.lrc`
- `fixtures/stage-demo-enhanced.lrc`
