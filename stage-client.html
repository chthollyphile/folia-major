Folia Stage API

Stage Controller Console

这个页面同时调试 Stage 的 HTTP 和 WS 协议。你可以推送会话、连接多个 Electron 实例、接收 control_request，并把同一份权威状态广播给多个 Folia。

Protocol Surface

GET /stage/health

无需鉴权。用于确认实例是否可访问，并查看当前 Stage 状态与 controller policy。
